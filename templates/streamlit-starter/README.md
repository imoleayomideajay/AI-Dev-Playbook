# Streamlit Starter

Beginner-friendly starter for an AI-assisted analytics app.

## Includes
- Clean page layout
- Sidebar controls
- Prompt input + placeholder AI response
- KPI cards
- Sample dataframe

## Run
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
