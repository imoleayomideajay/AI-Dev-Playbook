# Streamlit Starter Template

A minimal starter app for rapid AI-assisted dashboard and app prototyping.

## Features

- Clean Streamlit page structure
- Text input and submit interaction
- Placeholder AI response output
- Example dataframe rendering

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
streamlit run app.py
```

- Add input validation and error handling
